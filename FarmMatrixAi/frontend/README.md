# 🌾 FarmMatrixAI

FarmMatrixAI is a modern, scalable web application focused on clean UI, smooth user experience, and extensible architecture.  
The project is built using modern frontend tools and follows best practices for performance, maintainability, and design consistency.

---

## 🧰 Tech Stack

### ⚛️ Core
- **React** – Component-based UI
- **Vite** – Fast development & optimized builds
- **JavaScript (ES6+)**

---

## 🎨 UI & Styling

- **Tailwind CSS** – Utility-first CSS framework
- **DaisyUI** – Prebuilt, customizable Tailwind components
- **CSS** – Global and custom styles
- **Responsive Design** – Mobile-first approach

---

## 🎞️ UI Enhancements & Libraries

- **Keen Slider** – Touch-friendly sliders & carousels
- **Framer Motion** – Smooth animations & transitions
- **React Icons** – Lightweight icon library

---

## 🛠️ Development Tools

- **ESLint** – Code quality & consistency
- **Git & GitHub** – Version control
- **npm** – Dependency management

---

## ✨ Key Highlights

- Clean and scalable project structure  
- Reusable UI components  
- Smooth animations and sliders  
- Modern UI with DaisyUI themes  
- Easy to extend for future features (AI, backend, dashboards)
